1. O que é um SBGD?

Um SGBD (Sistema de Gerenciamento de Banco de Dados) é um software que serve como uma interface entre o usuário ou as aplicações e o banco de dados. Pense nele como um "bibliotecário digital" que não só armazena as informações, mas também as organiza, gerencia o acesso, garante a segurança e a integridade dos dados.

Exemplos: Lojas Virtuais (E-commerce) para informações de produtos, pedidos e transações.

2. Quais os principais bancos de dados?

Relacional: Oracle, MySQL, Microsoft SQL Server, PostgreSQL - Aplicações empresariais, financeiras, e-commerce, onde a integridade e a consistência dos dados são essenciais.

Não Relacional: MongoDB, Redis, Cassandra Big Data - Redes sociais, IoT, e aplicações que precisam de escalabilidade horizontal e flexibilidade de esquema.
