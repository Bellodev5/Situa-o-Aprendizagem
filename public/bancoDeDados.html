1. O que é um SBGD?

Um SGBD (Sistema de Gerenciamento de Banco de Dados) é um software que serve como uma interface entre o usuário ou as aplicações e o banco de dados. Pense nele como um "bibliotecário digital" que não só armazena as informações, mas também as organiza, gerencia o acesso, garante a segurança e a integridade dos dados.

Exemplos: Lojas Virtuais (E-commerce) para informações de produtos, pedidos e transações.

2. Quais os principais bancos de dados?

Relacional: Oracle, MySQL, Microsoft SQL Server, PostgreSQL - Aplicações empresariais, financeiras, e-commerce, onde a integridade e a consistência dos dados são essenciais.

Não Relacional: MongoDB, Redis, Cassandra Big Data - Redes sociais, IoT, e aplicações que precisam de escalabilidade horizontal e flexibilidade de esquema.

3. Dicionário de Dados e tipos de dados em um Banco de Dados

O dicionário de dados é um documento que descreve detalhadamente todas as tabelas, campos e tipos de dados de um banco de dados. Ele funciona como um manual técnico, facilitando o entendimento da estrutura e das relações entre as informações armazenadas.

Exemplo de Dicionário de Dados

Tabela Campo Tipo de Dado Tamanho Descrição Restrições
Aluno id_aluno INTEGER - Identificador único do aluno PK (chave primária)
Aluno nome VARCHAR 100 Nome completo do aluno NOT NULL
Aluno idade INTEGER - Idade do aluno CHECK (idade >= 0)
Aluno nota_avaliacao NUMERIC 4,2 Nota média do aluno -
Aluno id_matricula INTEGER - Identificador da matrícula FK (referência à tabela Matricula)

Tipos de Dados Comuns em Bancos de Dados

Categoria Tipo Descrição Exemplo
Numéricos INTEGER, BIGINT, NUMERIC, FLOAT Armazenam números inteiros ou decimais 25, 123456789, 12.50
Texto CHAR, VARCHAR, TEXT Armazenam caracteres ou textos longos 'Maria', 'Observações...'
Data/Hora DATE, TIME, TIMESTAMP Datas e horários 2025-10-09, 13:45:00
Lógicos BOOLEAN Valores verdadeiros ou falsos TRUE, FALSE
Outros SERIAL, UUID, JSON, BYTEA Tipos especiais para IDs, objetos e arquivos UUID, {"nome": "Carlos"}, imagem binária

Esses tipos de dados definem o formato e o comportamento das informações armazenadas em um banco de dados. A escolha correta de cada tipo garante melhor desempenho, integridade e organização dos dados.
