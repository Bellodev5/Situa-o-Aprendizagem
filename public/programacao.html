1. O que são Linguagens de Programação?

Linguagens de programação são ferramentas que permitem ao desenvolvedor instruir o computador a realizar determinadas tarefas.

Exemplos: C++, JavaScript, Java, Python, PHP.

2. Linguagem Interpretada vs Compilada

Interpretada: executada linha por linha por um interpretador (ex.: Python, JavaScript).

Compilada: traduzida para código de máquina antes da execução (ex.: C, C++).

3. O que é HTML e CSS?

HTML: estrutura do conteúdo da página.

CSS: define estilos e aparência.

4. O que é uma IDE?

Uma IDE (Ambiente de Desenvolvimento Integrado) é um software que reúne ferramentas para programar mais rápido e com menos erros.

Exemplos: VSCode, IntelliJ IDEA, Eclipse.
