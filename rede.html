Redes de Computadores

Entenda como computadores se comunicam, como a internet funciona e quais tecnologias tornam tudo isso possível.


Voltar ao Início

1. O que são Redes de Computadores?

É a interligação de dispositivos (computadores, celulares, servidores, etc.) que trocam informações entre si.

Exemplo: Rede doméstica, rede de uma empresa, internet.

2. O que é uma Rede Mínima?

É a menor rede possível: dois computadores conectados entre si, via cabo ou wi-fi.

3. O que é a Internet? O que a compõe?

A internet é uma rede mundial de computadores formada por:

  • Cabos submarinos
  • Servidores espalhados pelo mundo
  • Provedores de internet (ISP)
  • Roteadores e switches de grande porte

4. O que é Web?

É o conjunto de sites e páginas acessíveis via navegador (www). É uma parte da internet, não a internet inteira.

5. Mapa Global de Fibra Óptica

A fibra óptica conecta continentes e países e é responsável por mais de 95% do tráfego de internet do planeta.

6. Hardware de uma Rede Wi-Fi

  • Roteador Wi-Fi
  • Antenas
  • Modem
  • Access Points

7. Hardware de uma Rede Cabeada

  • Switch
  • Patch panel
  • Cabos Ethernet
  • Roteador
  • Racks

8. O que é um Protocolo de Comunicação?

É um conjunto de regras que define como dados são transmitidos em uma rede.

9. O que é o Modelo OSI?

É um modelo conceitual de 7 camadas que explica como a comunicação ocorre entre sistemas.

10. O Modelo TCP/IP

Modelo real usado na internet, composto por 4 camadas.

11. O que são IPv4 e IPv6?

IPv4: formato antigo (ex.: 192.168.0.1) — 4 bilhões de endereços.

IPv6: novo formato (ex.: 2804:348:...) — trilhões de endereços.

12. Como configurar IP/TCP em Node.js?

const http = require("http");

const server = http.createServer((req, res) => {
  res.end("Servidor funcionando!");
});

server.listen(3000, "127.0.0.1", () => {
  console.log("Rodando em http://127.0.0.1:3000");
});

13. O que é HTTP?

Protocolo usado para transmitir páginas web.

14. O que é HTTPS?

Versão segura do HTTP, utiliza criptografia TLS.

15. Exemplo de Framework HTTP em Node.js

Express.js — o mais usado.

const express = require("express");
const app = express();

app.get("/", (req, res) => {
  res.send("Olá mundo!");
});

app.listen(3000);
