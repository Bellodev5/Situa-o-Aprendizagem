Fundamentos da Tecnologia da Informação

Infraestrutura de TI (hardware, software, redes, gerenciamento de dados e serviços), a segurança da informação (confidencialidade, integridade e disponibilidade) e o processamento de dados para gerar informação e conhecimento estratégico para a tomada de decisões.

1. O que é Hardware? Placa-mãe, CPU e RAM

Hardware é toda a parte física do computador: peças e componentes palpáveis (placa-mãe, processador, memória, discos, fontes, periféricos).

Placa-mãe (Motherboard): placa principal que conecta todos os componentes, fornece soquetes (CPU, RAM), barramentos (PCIe), controladores e interfaces (USB, SATA).

Processador (CPU): unidade central de processamento — executa instruções, realiza cálculos e coordena tarefas do sistema. Características importantes: número de núcleos, frequência (GHz), cache.

Memória principal (RAM): memória volátil usada para armazenar temporariamente dados e instruções de programas em execução. A velocidade e a quantidade de RAM influenciam no desempenho multitarefa.

Observação: ao desligar o computador, o conteúdo da RAM é perdido (não persistente).

2. O que é um Sistema Operacional?

Sistema Operacional (SO) é o software base que gerencia hardware, recursos, processos, memória e fornece interface para o usuário e para aplicações.

Funções principais: gerenciamento de processos, gerenciamento de memória, sistema de arquivos, controle de dispositivos, segurança e interface (gráfica ou texto).

Exemplos:

  • Windows (ex.: Windows 10/11)
  • Linux (ex.: Ubuntu, Debian)
  • macOS (Apple)
  • Android (em dispositivos móveis)

3. O que é o Sistema de Arquivos? Modo texto (cmd/terminal) e modo gráfico (explorer)

Sistema de Arquivos é a estrutura usada pelo SO para organizar, nomear e armazenar arquivos em um dispositivo de armazenamento (HD/SSD/pen-drive).

Exemplos de FS: NTFS, FAT32, exFAT (Windows), ext4 (Linux), APFS (macOS).

Modo Texto (CMD / Terminal)

Operações comuns feitas por comando — rápido, ideal para administração e scripts.


# Windows (cmd)
dir
cd pasta
mkdir nova_pasta
del arquivo.txt

# Linux / macOS (terminal)
ls -la
cd /caminho/da/pasta
mkdir nova_pasta
rm arquivo.txt

Dica: no terminal você pode automatizar tarefas com scripts (batch, bash).

Modo Gráfico (Explorer / Finder)

Interface visual com pastas, miniaturas, arrastar/soltar — mais amigável para usuários finais.

  • Explorar pastas com ícones e visualizações (detalhes, miniaturas).
  • Copiar/colar arquivos com atalhos (Ctrl+C / Ctrl+V).
  • Operações com menu de contexto (botão direito).

Use o modo gráfico para tarefas visuais e o modo texto para tarefas repetitivas ou automação.

4. O que é Software de Escritório? Exemplos e aplicações

Softwares de escritório (office) são ferramentas para criar documentos, planilhas, apresentações e outros conteúdos de produtividade.

Editor de Texto

Ex: Microsoft Word, LibreOffice Writer

Aplicações: redação de relatórios, cartas, TCCs, formatação e revisão de texto.

Planilha Eletrônica

Ex: Microsoft Excel, LibreOffice Calc

Aplicações: cálculos, gráficos, controle financeiro, análise de dados e tabelas.

Apresentações

Ex: Microsoft PowerPoint, LibreOffice Impress

Aplicações: aulas, apresentações corporativas, pitches, slides para reuniões.

Os editores de texto permitem criar documentos estruturados, como relatórios, contratos, trabalhos escolares e textos profissionais.

As planilhas eletrônicas são usadas para cálculos, fórmulas, análises de dados, orçamentos e gráficos.

Programas de apresentação são usados para criar slides profissionais, palestras, aulas e apresentações corporativas.
